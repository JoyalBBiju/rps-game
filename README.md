# Stone Paper Scissors Game 🎮

A fun and interactive Stone Paper Scissors game built with HTML, CSS, and JavaScript!

## Features

✨ **Interactive Gameplay**
- Play against the computer
- Real-time score tracking
- Beautiful emoji indicators for each choice

🎨 **User-Friendly Interface**
- Responsive design (works on mobile and desktop)
- Smooth animations and transitions
- Modern gradient styling
- Clear display of moves and results

🔄 **Game Features**
- Keep track of your wins and computer wins
- Display of both player and computer choices
- Draw detection
- Reset button to start a new game

## How to Play

1. Open `index.html` in your web browser
2. Click on Stone, Paper, or Scissors to make your move
3. The computer will randomly choose its move
4. The winner is determined by the classic rules:
   - Stone beats Scissors
   - Scissors beats Paper
   - Paper beats Stone
5. Scores are updated automatically
6. Click "Reset Game" to start over

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Game logic and interactivity
- `README.md` - Documentation

## Game Rules

- **Stone** ✂️ beats **Scissors**
- **Scissors** 📄 beats **Paper**
- **Paper** 🪨 beats **Stone**
- If both choose the same, it's a **Draw**

## Installation

No installation required! Simply:

1. Clone the repository
   ```bash
   git clone https://github.com/JoyalBBiju/rps-game.git
   ```

2. Navigate to the folder
   ```bash
   cd rps-game
   ```

3. Open `index.html` in your browser
