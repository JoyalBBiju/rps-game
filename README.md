# 🎮 Stone Paper Scissors Game

A fun and interactive **Stone Paper Scissors game** built using **HTML, CSS, and JavaScript**, with development support from **GitHub Copilot**.

This project was created to practice **frontend development, JavaScript game logic, DOM manipulation, and AI-assisted coding workflows**.

---

## 🚀 Features

### 🎯 Interactive Gameplay

* Play against the computer
* Real-time score tracking
* Random computer moves
* Instant result display (Win / Lose / Draw)

### 🎨 User-Friendly Interface

* Responsive design for desktop and mobile
* Smooth animations and transitions
* Modern gradient UI styling
* Emoji-based move indicators

### 🔄 Game Functions

* Player vs Computer gameplay
* Live score updates
* Draw detection
* Reset button to restart the game anytime

---

## 🕹️ How to Play

1. Open **index.html** in your browser
2. Choose:

   * 🪨 Stone
   * 📄 Paper
   * ✂️ Scissors
3. The computer randomly selects its move
4. The winner is decided using classic game rules:

   * 🪨 Stone beats ✂️ Scissors
   * ✂️ Scissors beats 📄 Paper
   * 📄 Paper beats 🪨 Stone
5. Scores update automatically
6. Use **Reset Game** to start a new round
